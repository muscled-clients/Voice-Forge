üöÄ Getting Started

VoiceForge API provides powerful speech-to-text capabilities with support for:

Audio file transcription
YouTube video transcription
Real-time WebSocket streaming
50+ languages supported

üîë Authentication

Include your API key in the Authorization header:

Authorization: Bearer

üìä Base URL

All API endpoints use this base URL:

https://api.voiceforge.ai

üé§ Audio File Transcription

Transcribe audio files (MP3, WAV, M4A, etc.)

Endpoint:
POST /api/v1/transcribe
cURL Example:
curl -X POST https://api.voiceforge.ai/api/v1/transcribe \
  -H "Authorization: Bearer " \
  -F "file=@audio.mp3" \
  -F "language=auto"
Response:
{
  "status": "success",
  "transcription": {
    "text": "Hello, this is the transcribed text.",
    "language": "en",
    "duration": 10.5,
    "confidence": 0.95,
    "word_count": 6
  }
}

üé¨ YouTube Video Transcription

Extract and transcribe audio from YouTube videos

Endpoint:
POST /api/v1/transcribe/youtube
cURL Example:
curl -X POST https://api.voiceforge.ai/api/v1/transcribe/youtube \
  -H "Authorization: Bearer " \
  -H "Content-Type: application/json" \
  -d '{"url": "https://youtube.com/watch?v=VIDEO_ID", "language": "auto"}'
Response:
{
  "status": "success",
  "transcription": {
    "text": "Full video transcript here...",
    "language": "en",
    "duration": 120.5,
    "confidence": 0.92,
    "segments": [...]
  }
}

‚ö° Real-time WebSocket Streaming

Stream audio in real-time and get instant transcription results

WebSocket URL:
wss://api.voiceforge.ai/ws/v1/transcribe?api_key=
JavaScript Example:
const ws = new WebSocket('wss://api.voiceforge.ai/ws/v1/transcribe?api_key=YOUR_API_KEY');

ws.onopen = () => {
  console.log('Connected to VoiceForge streaming');
  
  // Send audio configuration
  ws.send(JSON.stringify({
    type: 'config',
    sample_rate: 16000,
    language: 'auto'
  }));
};

ws.onmessage = (event) => {
  const result = JSON.parse(event.data);
  if (result.type === 'transcription') {
    console.log('Transcription:', result.text);
  }
};

// Send audio chunks (binary data)
function sendAudioChunk(audioBuffer) {
  ws.send(audioBuffer);
}
Response Format:
{
  "type": "transcription",
  "text": "partial transcription",
  "is_final": false,
  "confidence": 0.87,
  "start_time": 1.2,
  "end_time": 2.5
}
