Welcome to VoiceForge

VoiceForge is a powerful speech-to-text API that transforms audio into accurate text transcriptions. With support for 100+ languages and state-of-the-art accuracy, VoiceForge is perfect for applications ranging from voice assistants to meeting transcription.

Key Features

  • High Accuracy: 99.5% accuracy with advanced neural networks
  • Multiple Languages: Support for 100+ languages with automatic detection
  • Real-time Processing: Stream audio for live transcription
  • Speaker Diarization: Identify different speakers in conversations

🚀 Ready to get started?

Check out our Quick Start Guide to transcribe your first audio file in minutes.

Quick Start

Get up and running with VoiceForge in less than 5 minutes.

Step 1: Get Your API Key

Sign up for a free account and get your API key from the developer dashboard.

Get API Key →

Step 2: Install SDK (Optional)

Install our official SDK for your preferred language:

Python 
pip install voiceforge
Node.js 
npm install voiceforge
Go 
go get github.com/voiceforge/voiceforge-go

Step 3: Make Your First Request

Transcribe your first audio file with a simple API call:

Python Example 
import voiceforge

# Initialize client
client = voiceforge.Client(api_key="YOUR_API_KEY")

# Transcribe audio file
with open("audio.mp3", "rb") as audio_file:
    result = client.transcribe(
        file=audio_file,
        language="auto"  # Auto-detect language
    )

# Print transcription
print(result.text)
print(f"Language: {result.language}")
print(f"Confidence: {result.confidence}")
Copied to clipboard!

Authentication

Learn how to authenticate your API requests to VoiceForge.

API Key Authentication

VoiceForge uses API keys to authenticate requests. You can find your API key in the developer dashboard.

Include your API key in the Authorization header: 

Authorization: Bearer YOUR_API_KEY

Example Requests

cURL

curl -X POST https://api.voiceforge.ai/v1/transcribe \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: multipart/form-data" \
  -F "file=@audio.mp3"

JavaScript

const formData = new FormData();
formData.append('file', audioFile);

const response = await fetch('https://api.voiceforge.ai/v1/transcribe', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY'
  },
  body: formData
});

⚠️ Security Best Practices

  • • Never expose your API key in client-side code
  • • Use environment variables to store API keys
  • • Rotate your API keys regularly
  • • Use different API keys for development and production

Transcribe Audio API

Convert audio files to text with high accuracy.

Endpoint

POST https://api.voiceforge.ai/v1/transcribe

Request Parameters

Parameter Type Required Description
file File Yes Audio file to transcribe (MP3, WAV, M4A, etc.)
language String No Language code (e.g., "en", "es"). Use "auto" for detection.
diarize Boolean No Enable speaker diarization (default: false)
punctuate Boolean No Add punctuation (default: true)
